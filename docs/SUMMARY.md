# Thecore Documentation

**Thecore** is a Ruby on Rails framework built around modular _ATOMs_ (Atomic Components). The documentation is organised into three layers:

---

## Primary documentation

| Document | Purpose |
|---|---|
| [GUIDE.md](GUIDE.md) | Complete developer reference: philosophy, commands, model anatomy, deploy, CI/CD |
| [WALKTHROUGH.md](WALKTHROUGH.md) | Step-by-step tutorial: build a full Thecore app from scratch |

Start with **WALKTHROUGH.md** if you are new to Thecore. Use **GUIDE.md** as your day-to-day reference.
